chrome-browser-formula
==================

A SaltStack formula designed to install and configure the [Notepad++](https://notepad-plus-plus.org/). 

It is primarily expected that this formula will be run via [P3](https://www.plus3it.com/)'s "[watchmaker](https://watchmaker.readthedocs.io/en/stable/)" framework.

This formula is able to install the Notepad++ graphical editor application Windows Server operating environments:

## Available states

- [notepad-plusplus](#notepad-plusplus)
